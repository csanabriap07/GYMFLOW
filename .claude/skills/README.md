# Skills del proyecto

Carpeta de skills compartida (ver `AGENTS.md` → "Herramientas del agente").
Claude Code la usa de forma nativa; OpenCode la lee por compatibilidad.

Actualmente vacía. Cuando el equipo agregue una skill, crear una subcarpeta
`nombre-skill/SKILL.md` siguiendo el mismo formato que las skills públicas
(descripción de cuándo aplica + instrucciones).
